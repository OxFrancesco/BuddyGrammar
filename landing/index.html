Features

⌨

Global Hotkeys

Assign a unique keyboard shortcut to each prompt profile — trigger rewrites from any app.

🔑

Keychain Storage

Your OpenRouter API key is stored in the macOS Keychain — never written to disk in plain text.

📋

Replace or Copy

Choose to replace the selection in place or copy the rewritten text to the clipboard.

🧩

Custom Profiles

Create multiple rewrite profiles with custom prompts — grammar, tone, translation, anything.

♾️

Accessibility-First

Uses macOS Accessibility APIs to capture and replace text, with a clipboard fallback.

▪

Lives in Menu Bar

A lightweight status item shows live rewrite progress — no windows to manage.

How it works

1

Select text anywhere

Highlight text in any application — a browser, Notes, Slack, your editor.

2

Press ^⌥G

The default Grammar shortcut fires — BuddyGrammar grabs the selection.

3

AI rewrites it

The text is sent to OpenRouter (gpt-5.4-nano) and rewritten in seconds.

4

Fixed text appears

The corrected text replaces your selection — or lands in the clipboard.
